# Explanation

## 36. there was / were  there has / have been  there will be

Este tema corresponde a la Unidad 36 de Basic Grammar in Use.

### 1. There has been / There have been

(Present Perfect)

Forma

- **There has been** → singular
- **There have been** → plural

Estructura:

```
There has been + singular noun
There have been + plural noun
```

Uso

Se usa cuando hablamos de algo que **ha ocurrido o ha existido** en un período que conecta con el presente.

👉 Es como decir:  
"Ha habido…" / "Ha existido…"

#### 🔹 Ejemplos

✔ Singular

- There **has been** a problem.  
    → Ha habido un problema.
    
- There **has been** an accident.  
    → Ha habido un accidente.
    

✔ Plural

- There **have been** many changes.  
    → Ha habido muchos cambios.
    
- There **have been** a lot of people here.  
    → Ha habido mucha gente aquí.
    

Idea clave del Present Perfect

Se usa cuando:

- El tiempo **no está terminado**
- No decimos cuándo exactamente ocurrió
- O tiene relación con el presente

Ejemplo:

- There have been many accidents recently.  
    (recientemente → conexión con el presente)

### 2. There was / There were

(Simple Past)

Forma

- **There was** → singular
    
- **There were** → plural
    

Uso

Se usa para hablar de algo que **existió en el pasado**, en un tiempo terminado.

👉 Es como decir:  
"Hubo…" / "Había…"

#### 🔹 Ejemplos

- There was a party yesterday.  
    → Hubo una fiesta ayer.
    
- There were many people at the party.  
    → Había muchas personas en la fiesta.

### 🔥 Comparación clara

|Tiempo          |Forma               |Uso                              |Ejemplo                                |
|----------------|--------------------|---------------------------------|---------------------------------------|
|Present Perfect |There has/have been |Pasado conectado con el presente |There have been many problems recently |
|Simple Past     |There was/were      |Pasado terminado                 |There were many problems last year     |

Diferencia importante

❌ There were many problems last year.  
(año pasado → terminado)

✅ There have been many problems recently.  
(recently → período conectado con ahora)

### 3. There will be

(Future)

Forma

```
There will be + singular/plural noun
```

👉 "Habrá…"

No cambia para singular o plural.

#### 🔹 Ejemplos

- There will be a meeting tomorrow.  
    → Habrá una reunión mañana.
    
- There will be many people at the event.  
    → Habrá muchas personas en el evento.
    

Resumen Final

|Tiempo          |Singular       |Plural          |Traducción   |
|----------------|---------------|----------------|-------------|
|Pasado          |There was      |There were      |Hubo / Había |
|Present Perfect |There has been |There have been |Ha habido    |
|Futuro          |There will be  |There will be   |Habrá        |

# 🧠 Ejemplos comparativos

- There **was** a problem yesterday.
- There **has been** a problem recently.
- There **will be** a problem tomorrow.

¿Ves la diferencia?

👉 Ayer (terminado)  
👉 Recientemente (conectado con ahora)  
👉 Mañana (futuro)
