## üî• Singular: Is

### Near (Cerca)
THIS: Este - Esta - Esto

- This is a blue car.
- This is an apple.
- This is my friend John.

### Far (Lejos)
THAT: Eso - Esa - Ese

- That is a blue car.
- That is an apple.
- That is my friend John.

## üî• Plural: Are 

### Near (Cerca)
THESE: Estas - Estos

- These are two new cars.
- These are red apples.
- These are my friends.

### Far (Lejos)
THOSE: Esos - Esas

- Those are two new cars.
- Those are red apples.
- Those are my friends.
